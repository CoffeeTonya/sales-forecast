# 販売予測アプリ

Streamlitを使用した時系列販売予測ツールです。過去の売上データから将来の売上数量と税抜売上金額を予測します。

## 機能

- 📊 日別の売上数量と税抜売上金額の予測
- 📈 複数の予測モデルから選択可能
- 📉 実績と予測を視覚的に比較できるグラフ表示
- 📋 予測結果の日別データテーブル表示
- 💾 予測結果のCSV出力

## 対応している予測モデル

### 1. Prophet
Facebookが開発した時系列予測モデル。季節性やトレンドを自動的に検出し、週次・日次のパターンを考慮した予測が可能です。

### 2. ARIMA（Auto ARIMA）
自己回帰移動平均モデル。統計的に堅牢な予測が可能で、パラメータは自動的に最適化されます。

### 3. Linear Regression
線形回帰モデル。シンプルで理解しやすく、トレンドを考慮した予測が可能です。

## インストール方法

1. 必要なライブラリをインストールします：

```bash
pip install -r requirements.txt
```

## 使用方法

1. アプリを起動します：

```bash
streamlit run app.py
```

2. ブラウザが自動的に開きます（通常は http://localhost:8501）

3. サイドバーで以下の設定を行います：
   - CSVファイルのアップロード（またはデフォルトファイルを使用）
   - 予測モデルの選択
   - 予測期間の設定

4. 「予測を実行」ボタンをクリックして予測を開始します

5. 予測結果はグラフとテーブルで表示され、CSV形式でダウンロード可能です

## CSVファイルの形式

以下のカラムが必要です：

- `売上日付`: 日付（例: "2025年12月01日"）
- `売上数量`: 数量（数値）
- `税抜売上金額`: 金額（数値、カンマ区切り可）

## 注意事項

- 予測の精度は、データの量と質によって異なります
- より長期的な予測ほど不確実性が高くなります
- 季節性やトレンドが明確なデータほど、ProphetやARIMAモデルの精度が高くなります
